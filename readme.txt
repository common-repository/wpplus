== Description ==

= وردپرس پلاس =

افزونه آخرین مطلب ارائه شده توسط وردپرس پلاس

http://wpplus.ir

== Installation ==

1. به برگه 'افزونه‌ها' در مدیریت وردپرس رفته و بر روی 'افزودن' کلیک کنید
2. 'wpplus' را جستجو کنید
3. افزونه را نصب کرده و توضیحات صفحه‌ی مدیریت آن را مطالعه کنید
